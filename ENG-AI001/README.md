# ENG-AI001 - Legacy Dataset Profiling & Ingestion Pipeline

## Project Overview

This project was developed as part of the Ontographica Engineering Internship Program.

The goal is to analyze and check a legacy CSV dataset before changing it into a Knowledge Graph.

## Features

* CSV Data Ingestion
* Structural Profiling
* Schema Inspection
* Missing Value Detection
* Statistical Analysis
* Automated Report Generation

## Technologies Used

* Python 3
* Pandas
* Visual Studio Code

## Execution

Run the script using:

python3 csv_analyzer.py

## Output

The script creates:

* analysis_report.txt

This file includes the dataset structure, missing value analysis, and numerical statistics.
